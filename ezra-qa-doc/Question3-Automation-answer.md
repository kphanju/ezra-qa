# Part 3 — Automation

I have not used Playwright extensively, but I have hands-on experience with Selenium for UI testing and OkHttp3 for API testing. The automation in this submission is implemented using that stack (Kotlin + Cucumber + Selenium + OkHttp3) and follows the Page Object Model pattern.

A sample automation project is available here:
**https://github.com/kphanju/ezra-qa/blob/main/ezra-qa-doc/README.md**
